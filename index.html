Chaski Framework

Lightweight Framework for Wi-Fi Tethering in Android

Chaski is a minimalistic framework to enable communication between off-the-shelf Android devices via common Wi-Fi (802.11 Infrastructure Mode).

Documentation

You can find a detailed documentation on getting started with Chaski in the Wiki.
